KWND Creative is looking for dedicated web developers and designers who are interested in working with creative digital media webpages.

Front-End Web Developer

Reference Number: ABC01

This position involves building and updating UI element of digital websites. You will work with designers and other developers.

Salary: $75,000 – $90,000 annually
Reporting line: Lead Developer

Key Responsibilities

  1. Creating web pages using HTML and CSS
  2. Helping ensure websites display correctly on different devices
  3. Updating and maintaining existing website content

Requirements

Essentials

  • Knowledge of HTML and CSS
  • Understanding of responsive web design
  • Active listening and communicating

Preferables

  • Experience with JavaScript
  • Familiarity with GitHub

Digital Content Designer

Reference Number: DEF02

This position focuses on creating visual content for websites and online platforms.

Salary: $65,000 – $80,000 annually
Reporting line: Creative Director

Key Responsibilities

  1. Designing images and layouts for digital content
  2. Following design guidelines and branding
  3. Working with team to complete projects on time

Requirements

Essentials

  • Basic skills in web design tools and software
  • Creative approach to solving design problems
  • Ability to transform ideas into clear digital visuals

Preferables

  • Basic HTML and CSS knowledge
  • Experience with designing web content
